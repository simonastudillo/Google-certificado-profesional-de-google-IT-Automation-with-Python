# Poniendo en marcha tu Python

## Instroducción al curso
- Cuando trabajas en TI, ​las habilidades de programación informática abren ​una cantidad increíble de oportunidades.
- ​La representación diversa es muy importante para demostrar a ​la próxima generación que pertenecen a un campo y que pueden prosperar allí.
- Al final de este curso, ​podrá manipular archivos y ​procesos en el sistema operativo de su computadora.
- Aprenderemos expresiones regulares
- Saber usar la línea de comandos
- Procesar un montón de errores en un archivo de registro real y generar un informe de errores.
- Cómo ejecutar Python localmente ​y organizar y usar código en diferentes archivos de Python. 
- La ​configuración de su entorno de desarrollador es un paso clave ​para poder escribir e implementar potentes herramientas de automatización
- En este curso, utilizaremos Qwiklabs ​, que es un entorno que le permite ​probar su código en una máquina virtual Linux

## Bienvenido al curso
- Pre requisitos:
    - La sintaxis básica de Python (if, for, while, definición de funciones, clases y métodos)
    - Cómo usar las estructuras de datos más comunes (cadenas, listas, tuplas y diccionarios)
    - Cómo importar y usar módulos adicionales de Python
