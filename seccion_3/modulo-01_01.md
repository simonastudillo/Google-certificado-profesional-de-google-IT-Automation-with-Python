# Introducción al curso

## Introducción al curso
- Esta sección se basa en como realizar un seguimiente de las diferentes versiones de un código
- Esto se conoce como VCSS (Version Control System Software) y es una herramienta muy importante para cualquier programador.
- El más popular es Git y cuenta con una plataforma de colaboración llamada GitHub.
- El control de versiones permite a los programadores trabajar en equipo, mantener un historial de cambios y revertir a versiones anteriores si es necesario.
- Aprenderemos la funcionalidad principal de Git, para entender el cómo y por qué se usa en las organizaciones.
