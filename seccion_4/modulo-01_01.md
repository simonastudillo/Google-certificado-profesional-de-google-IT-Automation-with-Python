# Introducción al curso

## Introducción al curso
- Problemas comúnes:
    - Averiguar porqué un programa no está haciendo lo que se espera.
    - Averiguar porqué se bloquea o se queda atascado un programa.
    - Optimizar la memoria, refactorizar para que funcione más rápido o transmitir menos datos
- Se darán escenarios de problemas basados en la vida real y se enseñarán técnicas de resolución de problemas y depuración para resolverlos.
